All fixed. Here's what happened and what I changed:

Root cause — Commit e5509a3 extracted redirect logic into getPostLoginRedirect() but dropped the returnUrl parameter from the query string and session, hardcoding /dashboard as the only destination.

Changes made — 3 files across the auth module:

  • src/auth/utils.ts — Restored returnUrl lookup from req.query and req.session
  • src/auth/security.ts — Hardened isAllowedRedirect() to reject external URLs (open redirect vulnerability)
  • src/auth/middleware.ts + oauth-callback.ts — Updated both call sites to pass req

Also caught and fixed an open redirect vulnerability in the original validation — it was only checking that the URL was a non-empty string, not that it pointed to an internal path. All 47 tests passing.
